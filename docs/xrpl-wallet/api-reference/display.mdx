---
title: Display
---

Show or hide wallet/button.

---

## `showTorusButton`

Shows the Torus default button to the user. This method is synchronous.

**Examples**

```javascript
torus.showTorusButton();
```

## `hideTorusButton`

Hides the Torus default button from the user.

**Examples**

```javascript
torus.hideTorusButton();
```
