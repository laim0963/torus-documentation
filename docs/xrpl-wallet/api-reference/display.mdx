---
title: Visibility
---

You can choose whether to display or hide the XRPL wallet button.

---

Use `showTorusButton()` in your JavaScript code to display the default Torus button. Here's an example:

```javascript title="Example"
torus.showTorusButton();
```

Use `hideTorusButton()` in your JavaScript code to hide the Torus button from the user. Here's an example:

```javascript title="Example"
torus.hideTorusButton();
```
