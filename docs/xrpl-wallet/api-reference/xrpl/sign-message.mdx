---
title: Sign Message
---

To sign a message, you can use the `signMessage` method.

---

`signMessage`

### Parameters

`params`

| Name     | Type                | Description             |
| -------- | ------------------- | ----------------------- |
| `params` | `SignMessageParams` | Sign Message Parameters |

`SignMessageParams`

| Name      | Type     | Description                  |
| --------- | -------- | ---------------------------- |
| `message` | `string` | Message to sign              |
| `from`    | `string` | Address to sign message with |

### Reference

```typescript
async signMessage(params: SignMessageParams): Promise<{ signature: string }> {
    if (!this.isInitialized) throw new Error("Torus is not initialized");
    const signMessageRes = await this.provider.request<SignMessageParams, { signature: string }>({
        method: "xrpl_signMessage",
        params: { ...params },
    });
    return signMessageRes as { signature: string };
}
```

### Example

```typescript
const message = "test message";
const res = await torus?.signMessage({
  message,
  from: "xrpl address",
});
```
